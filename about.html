About the Portal

The Cloud Change Intelligence Portal is an event‑driven, cost‑conscious system that captures and surfaces security‑relevant changes in your AWS environment. By leveraging native AWS services like CloudTrail, EventBridge, Lambda and S3, it generates an immutable feed of actions— enriched with context and risk scores— and publishes them as a static site.

Design Philosophy

  • Minimal cost: Uses static hosting and serverless compute so you only pay for what you use.
  • High signal, low noise: Filters down to IAM, networking and storage actions that matter most.
  • Evidence first: Each record links back to the CloudTrail event so you can audit with confidence.
  • Secure by default: Implements least‑privilege IAM roles and does not expose raw payloads.

Extensibility

This project is a starting point. You can extend it with additional analytics, dashboards or integrate it with chat and notification systems. Because it’s static, hosting is essentially free— the intelligence pipeline does the heavy lifting behind the scenes.
